🚀 QUICK START (5 MINUTES):
═══════════════════════════════════════════════════════════════════════

1. Install dependencies:
   cd backend && npm install

2. Create backend/.env:
   MONGO_URI=mongodb://localhost:27017/music_website
   JWT_SECRET=your_secret_key
   EMAIL_USER=your_gmail@gmail.com
   EMAIL_PASSWORD=your_16_char_app_password
   PORT=1337

3. Create frontend/.env:
   VITE_API_URL=http://localhost:1337

4. Get Gmail App Password:
   - Go to myaccount.google.com > Security
   - Enable 2-Step Verification
   - Get App Passwords for Mail
   - Copy 16-character password

5. Start servers:
   Terminal 1: cd backend && npm start
   Terminal 2: cd frontend && npm run dev

6. Test:
   Go to http://localhost:5173/otp-verification

🔌 API ENDPOINTS:
═══════════════════════════════════════════════════════════════════════

POST /api/v1/auth/send-otp
├─ Body: { email, fullName }
└─ Response: { message, status, email }

POST /api/v1/auth/verify-otp
├─ Body: { email, otp, password }
└─ Response: { message, status, token, user }

POST /api/v1/auth/resend-otp
├─ Body: { email }
└─ Response: { message, status, email }

POST /api/v1/auth/login
├─ Body: { email, password }
└─ Response: { message, status, token }

═══════════════════════════════════════════════════════════════════════

Step 1: User enters email & name
   ↓

   ↓
Step 3: User enters OTP + password
   ↓
Step 4: Email verified, account created
   ↓
Step 5: JWT token generated, user logged in
   ↓
Step 6: Redirected to home page

→ Verify MongoDB connection
